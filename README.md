# MATLAB PCA人脸识别（ORL+Yale人脸库）

#### 介绍
该系统为基于MATLAB平台的PCA的人脸识别系统，可识别ORL和YALE人脸库，方法实现统一，包括GUI界面。另外可二次开发成摄像头的实时人脸系统，识别出库外人脸，可做成门禁系统，考勤系统，打卡签到系统。实现登记出勤，报警等。
